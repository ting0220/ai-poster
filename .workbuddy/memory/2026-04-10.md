## 2. 公开页与管理后台分离

### 改动概述
- 首页 `/` → 公开模板列表 + 生成海报（无需登录）
- `/admin/templates` → 模板管理 + `/admin/fonts` 字体管理（需登录）

### 文件改动

1. **app/page.tsx** - 首页改为公开模板列表
2. **app/templates/_components/TemplatesListView.tsx** - 添加 `publicMode` 属性
   - 公开模式：隐藏"新建模板"和"修改模板"按钮，显示更新时间
   - 管理模式：显示所有功能
3. **app/admin/templates/page.tsx** - 简化，复用 TemplatesListView，保留密码验证
4. **app/admin/_components/AdminLayout.tsx** - 左侧导航添加"生成海报"入口

### 最终结构
- `/` - 公开生成海报（无需登录）
- `/admin/templates` - 模板管理（需登录）
- `/admin/fonts` - 字体管理（需登录）

## 3. 统一左侧导航栏

### 改动
1. 创建 `app/templates/_components/PublicLayout.tsx` - 统一的导航组件
2. 修改 `app/layout.tsx` - 根 layout 使用 PublicLayout
3. 修改 `app/admin/layout.tsx` - 管理后台也使用 PublicLayout，通过 title 属性区分
4. 删除 `app/templates/[id]/generate/layout.tsx` 和 `GenerateLayout.tsx`（避免双层导航）

### 效果
- 所有页面统一使用左侧导航栏
- 标题不同：首页显示"海报生成系统"，管理后台显示"管理后台"
- 未登录用户点击"模板管理"或"字体管理"会跳转到密码验证页面
