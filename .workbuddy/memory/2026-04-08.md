## 技术细节

### 响应式布局关键代码
```jsx
// 预览宽度计算
const previewWidth = Math.floor(maxPreviewHeight / canvasRatio);
// 限制最大宽度
Math.min(calculatedWidth, Math.floor(window.innerWidth * 0.5))

// 布局类
<div className="flex flex-col md:flex-row md:items-start gap-6">
  {/* 预览区 - 小屏幕全宽，大屏幕固定宽度 */}
  <div className="md:shrink-0" style={{ width: `min(100%, ${previewWidth}px)` }}>
  {/* 输入区 - 自适应剩余空间 */}
  <div className="flex-1 md:min-w-0">
```

### 高度不足警告计算
- 行高 = 字号 × 1.4
- 所需高度 = 最大行数 × 行高
- 如果当前高度 < 所需高度，显示警告
